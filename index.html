SITUACION PROBLEMA

OBJETIVO GENERAL

Desarrollar el Sistema Gestor de Empleados y Contrataciones para la empresa CentroVenezolano C.A

OBJETIVOS ESPECIFICOS

  1. Analizar los procesos de contratación y asignación de permisos que se realizan dentro de la organización.
  2. Seleccionar las tecnologías de desarrollo a utilizar.
  3. Levantar la información referente a los requerimientos del sistema.
  4. Planificar iteraciones con entregas del sistema.
  5. Desarrollar el sistema de acuerdo a la planificación realizada.
  6. Realizar las pruebas para la depuración del sistema.
  7. Documentar el sistema.

METODOLOGIA Y TECNICAS APLICADAS

PROGRAMACION EXTREMA (XP)

ROLES EN XP

  • Programador
  • Cliente
  • Tester
  • Tracker
  • Entrenador
  • Consultor
  • Gestor

CICLO DE VIDA DE XP

EXPLORACION

En esta fase los clientes realizan las historias de usuario y los desarrolladores definen las herramientas a utilizar

PLANIFICACION

Durante esta fase se refinan, ordenan y clasifican las historias de usuarios realizadas por el cliente

ITERACION

En esta fase se decide el numero de iteraciones necesarias para culminar el desarrollo del sistema

PRODUCCION

En esta fase el producto sale de un ambiente de desarrollo a uno de produccion.

MANTENIMIENTO

Durante esta fase se pueden agregar nuevas funcionalidades al sistema

MUERTE

Esta fase define la culminacion del proyecto

IMPLEMENTACION DE LA METODOLOGIA

ROLES

  • Programador: Pasante
  • Cliente: Usuario final
  • Tester: Gerente de sistemas y vicepresidente de CentroVenezolano C.A.
  • Entrenador y Tracker: Gerente de sistemas.
  • Consultor: Analistas de sistemas.
  • Gestor: Vicepresidente.
